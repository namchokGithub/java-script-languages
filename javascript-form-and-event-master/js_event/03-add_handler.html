03 - Add handler

event listenter (ตัวฟังเหตุการณ์)

การเพิ่ม Event มี method คือ addEventListener ( "eventName", function_name, capture );

ตัวแปร capture ถ้ากำหนด true คือ จะโหลด function ไว้ที่หน่วยความจำเพื่อรอการเรียกใช้ตั้งแต่การโหลด page แต่ถ้ากำหนด false คือ จะโหลด function เฉพาะตอนที่มีการเรียกใช้ event นั้นๆ
